# Drone Delivery Service

## Coding Challenge

A squad of drones is tasked with delivering packages for a major online retailer in a
world where time and distance do not matter.

## Requirements

Java8+
Gradle (optional)

## How to run

### Using Gradle

Run `./gradlew run`

to run the test

Run `./gradlew test`

### Without Gradle

#### Linux
Run `./run.sh`

#### Windows

Run `./run.bat`

### Sample files
Sample files are located in the samples directory
